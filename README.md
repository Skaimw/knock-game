# Knock — Card Game Client

The complete Knock card game as a single HTML file.

## Play
Open `index.html` in any browser — works standalone with AI opponents.

## Multiplayer
Add `?server=https://your-server.up.railway.app` to the URL to enable
real-time multiplayer via the knock-server backend.

## Deploy
This repository is auto-deployed to Vercel on every push to `main`.
